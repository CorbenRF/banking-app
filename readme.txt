1. Для запуска сервера бэкенда: из папки с сервером npm start

2. Для сборки dev: npm run dev

3. Для сборки dev: npm run build

3.1. После сборки запустить лайв сервер командой serve <путь к серверу> -s (флаг -s нужен для правильной работы роутера в режиме SPA)

4. Для сборки jest тестов: npm run test (тестирует обработку сервером балансов из прошлого на примере готового объекта.
Тот же объект есть и в файле index.js в пути роутера с детальной историей. Если его раскомментировать и закомментировать обращение к серверу,
то можно увидеть как строятся графики с разнообразной историей транзакций)

5. Для тестирования Cypress - установить пакет cypress и запустить файл index.spec.cy.js
